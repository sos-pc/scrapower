🧩 Intégrer Scrapower sur votre site

Ajoutez cette iframe n'importe où sur votre page. Le widget charge un worker de calcul distribué. Aucun accès à votre DOM. Le visiteur doit cliquer "Activer" pour commencer. 

<iframe
  src="https://your-coordinator.example.com/embed"
  style="width:260px; height:85px; border:none;
         position:fixed; bottom:16px; right:16px;
         z-index:9999; border-radius:8px;
         overflow:hidden;"
  title="Scrapower — Calcul distribué"
  loading="lazy">
</iframe>
✅ Aucune dépendance JavaScript
✅ Isolé dans une iframe (sandbox)
✅ Opt-in : le visiteur clique pour activer
✅ Conforme RGPD : pas de cookies, pas de tracking
